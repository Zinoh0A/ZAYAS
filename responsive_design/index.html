ZAYAS Responsive Design Documentation

Comprehensive guide to responsive design implementation across all pages and device types

Overview

This documentation provides a comprehensive guide to the responsive design implementation in the ZAYAS e-commerce project. The responsive design ensures that the website looks and functions well across different device types and screen sizes.

Breakpoints

Name Size Range Typical Devices
Extra Small (xs) < 576px Mobile phones (portrait)
Small (sm) ≥ 576px Mobile phones (landscape)
Medium (md) ≥ 768px Tablets
Large (lg) ≥ 992px Laptops/Desktops
Extra Large (xl) ≥ 1200px Large desktops

Responsive Components

Key components that adapt across different screen sizes:

Navigation Header

Transforms from horizontal menu to hamburger menu

Product Cards

Adjust from 4-per-row to 1-per-row

Cart Table

Transforms to card view on mobile

Admin Dashboard

Collapsible sidebar and stacked cards

Checkout Form
